# IPL Cricket Analysis Dashboard 2008-2022

## Project Overview
Analyzed 950+ IPL matches from 2008 to 2022 using Microsoft Excel.

## Key Findings
- Mumbai Indians — most wins (131 matches)
- AB de Villiers — top Player of Match (25 times)
- Matches grew from 58 to 76 per season
- Toss winner doesn't always win the match!

## Tools Used
- Microsoft Excel
- Pivot Tables
- Interactive Slicers
- Data Visualization

## Dataset
Kaggle — IPL Matches 2008-2022
